# Contributing to Solana Whale Monitor

Thank you for your interest in contributing to this on-chain analytics bot.

## Workflow

1. Fork the repository and create your branch from `main`.
2. Ensure you have tested your bot logic against a local mock of the Telegram API or a private test bot.
3. Submit a pull request.

## Pull Request Process

- Keep commits atomic and descriptive (e.g., `feat: add OKX datapi fallback`).
- Do not commit any `.env` files or API keys.
